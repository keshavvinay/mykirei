G_NAV_004: Brand Footer

BrandFooter

  • length: Sitemap内の Unitの数
  • Rows(lg): 1-5
  • Rows(md): 1-4
  • Element: Logo(DOM出力変更)、Sitemap
  • Logo Width : 10 - 100% (opt class)
    （ロゴ画像は 幅160px 高さ90px を最大サイズとして、そのサイズに収まる範囲内で縦横比率を保持して表示 ）
  • Logo Align (lgのみ) : center(default), left, right
  • Logo VerticalAlign (lgのみ) : top(default), middle, bottom
  • add parsys: y/n
  • Background image : 背景画像スタイル付与
  • Specific CSS class : 任意クラス付与
  • ID : id付与

Default: length 4、Rows(lg) 4、Rows(md) 2、ロゴ無し
