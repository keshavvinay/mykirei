  • Type : タグ要素 h2-h6,p
  • Element : Sub Title(Top), Sub Title(Bottom), Sub Title(Right), Main Image（DOM出力変更）, Sub Image（DOM出力変更）
  • Link : div → aタグ変更
  • Link icon : 保留
  • Text align : Text-align制御クラス付与
  • Layout : 画像左、画像右(DOM出力順変更、制御クラス付与)
  • Main Image-Width : 100%〜10%（bpごとにoptionクラス付与）
  • Sub Image-Width : 50%〜5%（bpごとにoptionクラス付与）
  • Specific CSS class : 任意クラス付与
  • Background image : 背景画像スタイル付与
